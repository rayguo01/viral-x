# Content Writer Skill

基于 Defou x Stanley 方法论的内容创作工具。

## 功能

输入原始素材/想法，自动生成三个版本的内容：

- **🔥 版本 A (Stanley Style)**：追求极致的点击率和传播度，情绪饱满，金句频出
- **🧠 版本 B (Defou Style)**：侧重底层逻辑拆解和深度认知，提供长期价值
- **🌟 版本 C (Combo Style)**：Defou x Stanley 终极融合版，结合传播节奏与深度内核

## 输出

- **最终版本**：版本 C（融合版）
- **智能评估**：好奇心、共鸣度、清晰度、传播值四维评分

## 使用方式

在 Web UI 工具栏点击「内容创作」按钮，输入素材后生成报告。

## 输出目录

- 报告文件：`outputs/content/content_[timestamp].md`
