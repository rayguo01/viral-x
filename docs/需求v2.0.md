# 项目需求
制作一个x平台帖子生成器。
运用现在已有的skill，完成整个帖子自动生成的过程

x帖子生成流程：
首页是生成流程的节点图：
|热帖抓取|生成内容|优化内容|生成图片|提交到X平台
- 热帖抓取：x-trends，tophub-trends
- 生成内容：content-writer
- 优化内容：viral-verification
- 生成图片：gemini-image-gen
- 提交到X平台: 还未实现，先不实现

用户访问首页，首先看到的就是这个生成流程图，用户点击x-trends或tophub-trends节点，进入 X帖子生成引导器，每个环节是一个页面，比如点x-trends，出现最新的话题趋势，用户选择一个话题趋势，会进入到第二页content-writer，点击 编写帖子 按钮，会生成内容，展示给用户，包括：版本 C内容（可编辑），版本 C 评分，以及优化建议；用户点击优化按钮，进入第3页，这里会展示爆款要素优化报告内容，其中 最终优化爆款版本 可编辑；在用户完成编辑后，点击确认，进入到下一页，系统根据帖子内容，生成一个生成图片的prompt（可编辑），图片比例是适合x平台发送的；用户可以选择生成图片，或者跳过该步骤，进入下一页提交到X平台，在该页面，显示最终提交的帖子内容；提交的图片（可选），一个提交按钮

这就是用户的主流程，让用户可视化，直观的了解帖子的生成步骤
